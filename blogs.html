What are the different ways to select an element in the DOM?

By using getElementById
const elementById = document.getElementById("myElement");
By using querySelector
const elementByQuerySelector = document.querySelector(".myClass");
By using querySelectorAll
const elementsByQuerySelectorAll = document.querySelectorAll("div.myClass");
By using getAttribute
const attributeValue = document.querySelector("a").getAttribute("href");
By using closest
const closestElement = document.querySelector(".child").closest(".parent");

What is the difference between innerHTML, innerText, and textContent?

innerHTML innerText textContent
Represents the HTML content inside an element, including HTML tags. Represents the visible text content of an element, considering the rendered output. Represents the raw text content of an element, excluding HTML tags.
Includes HTML tags (e.g., <p>, <div>). Excludes HTML tags but respects visible content (e.g., hidden or styled elements are excluded). Excludes HTML tags and includes all text (even hidden text).
Does not consider CSS visibility or styling. Considers CSS styling (e.g., visibility, text-transform). Non-visible text is excluded. Does not consider CSS styling. All text is included regardless of visibility.
Relatively slower because it involves parsing HTML content. Slower than textContent due to layout reflows for rendering the visible text. Faster than innerText as it does not require rendering or layout calculations.

What is event delegation in the DOM?

Event delegation is a technique in JavaScript where you attach a single event listener to the parent element instead of attaching numerous event listeners to its child elements. This takes advantage of event bubbling, where an event on a child element bubbles (or propagates) to its parent elements.

What is event bubbling in the DOM?

Event bubbling is an aspect of the DOM (Document Object Model) where an event, triggered on a child object, "bubbles up" to its parents within the DOM tree. So when a mouseover, a click, or some other type of event is triggered on an object, it does not merely interact with the elementâ€”it finds its way back up through every one of its parents until it reaches the top of the tree (the document object).

How do you create, add, and remove elements using JavaScript?

To create an element
let newElement = document.createElement('div'); // Creates a new div element
To add an element to the DOM
document.body.appendChild(newElement); // Adds the new element to the body
To Remove an element from the DOM
newElement.remove(); // Removes the new element
